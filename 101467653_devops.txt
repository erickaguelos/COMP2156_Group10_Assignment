What is DevOPs?
  -A combination of software development(dev) and 
  operations (ops)
  -Represents the current stat of the evolution of software
  delivery cycles during past 20+ years.
  -Ultimately, DevOps is about meeting software users ever-increasing
  demand for frequent, innovative new features and uninterrupted performance and 
  availability. 
  -  DevOps practices typically include automation, continuous integration/continuous delivery (CI/CD), infrastructure as code (IaC), monitoring, and feedback loops to enable 
  faster and more reliable software development and deployment.
